1. 依存関係インストール

```
bun install
```

2. データベース設定

このプロジェクトでは、SQLite と Drizzle ORM を使用しています。
ローカルの SQLite データベースを起動します（任意）
D1 のローカル開発とマイグレーションは、開発時およびデプロイ時に Alchemy によって自動的に処理されます。
必要に応じて、apps/web ディレクトリ内の .env ファイルに適切な接続情報を設定してください。

```
# スキーマをデータベースへ適用
bun run db:push
```

3. 開発サーバーを起動

```
bun run dev
```

ブラウザで http://localhost:3001 を開くとアプリを確認できます
